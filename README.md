# UniteChan

ポケモンユナイト向けの Discord Bot。ロビー管理・チーム分け・VC移動・ロール/ポケモン自動割当をサポートします。

## 機能

- **ロビー管理** — メンバーの参加/離脱・ランク登録・VC一括収集・管理者による代理追加
- **チーム分け** — ランク/戦績バランス・ロール割当・ポケモン割当を5桁コードで制御
- **VC自動移動** — チーム分け結果に従ってメンバーをVCに振り分け（リアクション操作対応）
- **前回チーム分散** — 連続で同じチームにならないよう自動で分割
- **特定ペア分離** — 指定した2人を必ず別チームにする
- **ポケモンバン** — 使用させたくないポケモンをサーバーごとに設定
- **連続ロール回避** — 同じロールが続かないよう履歴を考慮した割当
- **戦績管理** — リアクションまたはコマンドで試合結果を記録・通算/当日ランキング表示
- **表示名変更** — ボット側のみ表示名を変更（サーバーニックネームは変わらない）

## コマンド一覧

### ロビー

| コマンド | 説明 |
|---|---|
| `/join` | 自分がロビーに参加 |
| `/join <メンバー>` | 指定メンバーをロビーに追加 ★管理者 |
| `/leave` | ロビーから抜ける |
| `/rank` | 自分のランクを登録・変更 |
| `/lobby` | 現在のロビーメンバーを表示 |
| `/lobby_collect` | 参加中のVCメンバーをロビーに一括登録 ★管理者 |
| `/lobby_clear` | ロビーを全員解散 ★管理者 |
| `/kick <メンバー>` | メンバーをロビーから追放 ★管理者 |
| `/name [name] [member]` | ボット上の表示名を変更（省略でリセット、member指定は管理者専用） |

### チーム分け

| コマンド | 説明 |
|---|---|
| `/split run` | チーム分けを実行（コードは `/config split` で設定） |
| `/split test [モード]` | デモデータでチーム分けをテスト ★管理者 |
| `/split move [channel_a] [channel_b]` | チーム分け結果に従いVCにメンバーを移動 ★管理者 |

> **リアクション操作:** `/split run` 後のメッセージに以下のリアクションが自動付与されます。
>
> | リアクション | 動作 |
> |---|---|
> | 🇦 / 🇧 | そのチームの勝利として試合結果を記録 |
> | 🎙️ | `/config vc` のVCに全員を一括移動 ★管理者 |
> | 🔄 | 同じ設定でリロール（メッセージを更新） ★管理者 |

### バン

| コマンド | 説明 |
|---|---|
| `/ban add <ポケモン>` | ポケモンをバン ★管理者 |
| `/ban remove <ポケモン>` | バンを解除 ★管理者 |
| `/ban list` | バン中ポケモン一覧を表示 |
| `/ban clear` | 全バンを解除 ★管理者 |

### 戦績

| コマンド | 説明 |
|---|---|
| `/result <team>` | 試合結果を手動で記録（Team A / Team B） |
| `/result_undo` | 直前の試合結果記録を取り消し ★管理者 |
| `/stats show [member] [period]` | 戦績ランキングまたは個人の戦績を表示（period: 通算 / 当日） |
| `/stats reset` | 通算・当日の戦績をすべてリセット ★管理者 |

> 当日戦績は **JST 05:00** にリセットされます。

### 分離設定

| コマンド | 説明 |
|---|---|
| `/separate add <member1> <member2>` | 2人を必ず別チームに設定 ★管理者 |
| `/separate remove <member1> <member2>` | 分離設定を解除 ★管理者 |
| `/separate list` | 分離設定一覧を表示 |
| `/separate clear` | 全分離設定を解除 ★管理者 |

### 設定

| コマンド | 説明 |
|---|---|
| `/config split <コード>` | デフォルトのチーム分けコードを設定 ★管理者 |
| `/config role_balance` | ロール構成を設定（ATK/ALL/SPD/DEF/SUP 各人数） ★管理者 |
| `/config avoid <回数>` | 連続ロール回避の回数を設定（0〜5） ★管理者 |
| `/config vc <team> <channel>` | `/split move` や 🎙️ リアクションで使うデフォルトVCを設定 ★管理者 |
| `/config show` | 現在の設定を表示 |
| `/config reset` | 設定をリセット ★管理者 |

## チーム分けモードコード

`/split run` のコードは `/config split` でサーバーごとに保存できます。5桁の数字で指定します。

```
[a][b][c][d][e]
```

| 桁 | 意味 | 値 |
|---|---|---|
| a | バランス方式 | `0` = ランダム / `1` = ランクで均等分け / `2` = 通算勝率で均等分け / `3` = 当日勝率で均等分け |
| b | ロール割当 | `0` = なし / `1` = 自動(各1枚) / `2` = `/config role_balance` の設定値 |
| c | ポケモン割当 | `0` = なし / `1` = 個人割当 / `2` = チーム割当 |
| d | 連続ロール回避 | `0` = OFF / `1` = ON（`/config avoid` の回数を使用） |
| e | チーム間重複許可 | `0` = 禁止 / `1` = 許可 |

**例:**

| コード | 意味 |
|---|---|
| `00000` | 完全ランダム |
| `10000` | ランクバランスのみ |
| `11100` | ランクバランス + ロール自動 + ポケモン割当 |
| `12110` | ランクバランス + ロール設定値 + ポケモン割当 + 連続回避 |
| `20000` | 通算勝率バランス |
| `30000` | 当日勝率バランス（05:00リセット） |

## VC移動の流れ

1. `/lobby_collect` または `/join` でロビーを組む
2. `/split run` でチーム分けを実行
3. 🎙️ リアクションまたは `/split move` でメンバーを Team A / Team B のVCに自動移動

デフォルトVCを `/config vc` で設定しておけば引数なしで移動できます。

## セットアップ

### 必要なもの

- Python 3.11+
- Discord Bot Token

### インストール

```bash
git clone https://github.com/LaceCelestia/UniteChan.git
cd UniteChan
pip install -e .
```

### 設定

`.env` ファイルを作成:

```env
DISCORD_TOKEN=your_token_here
```

### 起動

```bash
# Windows
run.bat

# または直接
python -m unitechan
```

## ポケモンリスト

`data/pokemon_list.yaml` にロール別のポケモンリストが定義されています。追加・削除することでバン候補や割当対象を変更できます。

```yaml
attacker:
  - ピカチュウ
  - ゲッコウガ
  # ...
speedster:
  - ゼラオラ
  # ...
```

## データ永続化

| ファイル | 内容 |
|---|---|
| `data/lobby_state.json` | ロビーメンバー・ランク・表示名エイリアス |
| `data/config_state.json` | サーバー設定・バン中ポケモン・分離ペア |
| `data/stats_state.json` | 通算戦績・当日戦績・直前の試合結果 |
