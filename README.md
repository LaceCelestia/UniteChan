# UniteChan

ポケモンユナイト向けの Discord Bot。ロビー管理・チーム分け・ロール/ポケモン自動割当をサポートします。

## 機能

- **ロビー管理** — メンバーの参加/離脱・ランク登録・VC一括収集
- **チーム分け** — ランクバランス・ロール割当・ポケモン割当を5桁コードで制御
- **前回チーム分散** — 連続で同じチームにならないよう自動で3:2分割
- **ポケモンバン** — 使用させたくないポケモンをサーバーごとに設定
- **連続ロール回避** — 同じロールが続かないよう履歴を考慮した割当
- **戦績管理** — 試合結果を記録し勝敗ランキングを表示

## コマンド一覧

### ロビー

| コマンド | 説明 |
|---|---|
| `/join` | ロビーに参加 |
| `/leave` | ロビーから抜ける |
| `/rank` | 自分のランクを登録・変更 |
| `/lobby` | 現在のロビーメンバーを表示 |
| `/lobby_collect` | 参加中のVCメンバーをロビーに一括登録 ★管理者 |
| `/lobby_clear` | ロビーを全員解散 ★管理者 |
| `/kick <メンバー>` | メンバーをロビーから追放 ★管理者 |

### チーム分け

| コマンド | 説明 |
|---|---|
| `/split run <モード>` | 5桁コードでチーム分けを実行 |
| `/split test <モード>` | デモデータでチーム分けをテスト ★管理者 |

### バン

| コマンド | 説明 |
|---|---|
| `/ban add <ポケモン>` | ポケモンをバン ★管理者 |
| `/ban remove <ポケモン>` | バンを解除 ★管理者 |
| `/ban list` | バン中ポケモン一覧を表示 |
| `/ban clear` | 全バンを解除 ★管理者 |

### 戦績

| コマンド | 説明 |
|---|---|
| `/result <team>` | 試合結果を記録（Team A / Team B） |
| `/stats show [member]` | 戦績ランキングまたは個人の戦績を表示 |
| `/stats reset` | 戦績をすべてリセット ★管理者 |

### 設定

| コマンド | 説明 |
|---|---|
| `/config role_balance` | ロール構成を設定（ATK/ALL/SPD/DEF/SUP 各人数） ★管理者 |
| `/config avoid <回数>` | 連続ロール回避の回数を設定（0〜5） ★管理者 |
| `/config show` | 現在の設定を表示 |
| `/config reset` | 設定をリセット ★管理者 |

## チーム分けモードコード

`/split run` には5桁の数字コードを指定します。

```
[a][b][c][d][e]
```

| 桁 | 意味 | 値 |
|---|---|---|
| a | ランクバランス | `0` = ランダム / `1` = ランクで均等分け |
| b | ロール割当 | `0` = なし / `1` = 自動(各1枚) / `2` = `/config` の設定値 |
| c | ポケモン割当 | `0` = なし / `1` = 個人割当 / `2` = チーム割当 |
| d | 連続ロール回避 | `0` = OFF / `1` = ON (`/config avoid` の回数を使用) |
| e | チーム間重複許可 | `0` = 禁止 / `1` = 許可 |

**例:**

| コード | 意味 |
|---|---|
| `00000` | 完全ランダム |
| `10000` | ランクバランスのみ |
| `11100` | ランクバランス + ロール自動 + ポケモン割当 |
| `12110` | ランクバランス + ロール設定値 + ポケモン割当 + 連続回避 |

## セットアップ

### 必要なもの

- Python 3.11+
- Discord Bot Token

### インストール

```bash
git clone https://github.com/LaceCelestia/UniteChan.git
cd UniteChan
pip install -e .
```

### 設定

`.env` ファイルを作成:

```env
DISCORD_TOKEN=your_token_here
```

### 起動

```bash
# Windows
run.bat

# または直接
python -m unitechan
```

## ポケモンリスト

`data/pokemon_list.yaml` にロール別のポケモンリストが定義されています。追加・削除することでバン候補や割当対象を変更できます。

```yaml
attacker:
  - ピカチュウ
  - ゲッコウガ
  # ...
speedster:
  - ゼラオラ
  # ...
```

## データ永続化

| ファイル | 内容 |
|---|---|
| `data/lobby_state.json` | ロビーメンバー・ランク情報 |
| `data/config_state.json` | サーバー設定・バン中ポケモン |
| `data/stats_state.json` | 戦績・直前の試合結果 |
